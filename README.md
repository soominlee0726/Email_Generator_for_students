# LLM-based Academic Email Generator

This project is a simple prototype that generates polite academic emails from student inputs using instruction prompting and few-shot learning.

## Features
- Converts informal user input into formal academic email
- Uses instruction-tuned LLM (Qwen)
- Few-shot prompting for improved quality

## Example

**Input(in Korean)**
Write a polite email to Professor Soo-min Lee about missing class next week.

**Output(in Korean)**
(Generated email)

## Conclusion
Few-shot prompting was more effective than small-scale fine-tuning under limited data conditions.
